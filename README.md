## Usage
1. Set up the virtual environment and install requirements

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Running, Linting, and Testing the program

Run: `make run`
Lint: `make lint`
Test: `make test`
Submit: `make submit`
